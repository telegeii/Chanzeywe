# Chanzeywe TVC — Official Website

> Skills to Transform Livelihoods — Vihiga County, Kenya

A full-stack college website for Chanzeywe Technical and Vocational College built with React, Node.js, Express and MySQL.

🔗 **Live:** https://chanzeywe.up.railway.app/
📁 **GitHub:** https://github.com/telegeii/Chanzeywe

---

## Tech Stack

- **Frontend:** React 19, React Router, Vite
- **Backend:** Node.js, Express, MySQL
- **Hosting:** Railway

---

## Features

- Course listings and online student application form
- Tenders, careers, downloads, and blog/news pages
- Anonymous corruption reporting (whistleblower protected)
- Admin dashboard to manage all content and applications

---

## Getting Started

```bash
# Clone
git clone https://github.com/telegeii/Chanzeywe.git
cd Chanzeywe

# Install & run frontend
npm install
npm run dev

# Install & run backend
cd backend
npm install
cp .env.example .env   # fill in DB credentials
npm run dev
```
